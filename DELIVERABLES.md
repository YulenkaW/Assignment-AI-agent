# Goal-Driven Project Deliverables Checklist

## Assignment Deliverables

- GitHub repository with working code and a clear README
- demo video with:
  - one understanding query
  - one build or execution query
- design document covering:
  - architecture
  - context strategy
  - design decisions

## Ground Rules To Respect

- use a local pre-downloaded `nlohmann/json` repository
- do not clone or download repositories from the agent
- execute commands for real on the local machine
- enforce the 5,000-token limit programmatically
